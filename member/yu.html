Git 教學

Git是什麼？ git是一種版本控制系統，同樣為版本控制的還有SVN 透過這樣的版本控制系統，可以清楚的記錄每個檔案是誰在什麼時候加進來、什麼時候被修改或刪除。

  1. 建立新的目錄，並且Git初始化
  2. mkdir 資料夾名稱
  3. cd 資料夾名稱
  4. git init
  5. git clone 你的專案網址

好了之後輸入

  • 查看目前檔案的狀態
    • git status
  • 將檔案提交入Repository
    • git commit -m "解說文字"
  • 把所有更改過的東西都 add
    • git add .

    提交完之後再用git status看一次，你的狀態會變成Changes to be committed

  • 推送到Github上面
    • git push

    終端機文件修改常用指令

    1. :i 編輯文件
    2. :w 存擋
    3. :q 離開
    4. :wq 存檔離開
    git log 紀錄檔/git diff 比較 git log//按q可跳出瀏覽 git rm git rm 將檔案刪除 git mv git mv 檔案名 資料夾 搬移檔案，這樣就不用add或rm檔案了 git rebase git rebase 將不同分支上的差異點合併成一個新的commit git config git config --global 設定初始參數 git config --list 可查看設定的參數
